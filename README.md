# proxydesignpattern
JDK动态代理中要求目标类实现的接口数量不能超过65535个，是因为在tcp udp协议的开头，会分别有16位来存储源端口号和目标端口号，所以端口个数是2^16-1=65535个，所以在Proxy中的getProxyClass0这个方法中会做接口限制
